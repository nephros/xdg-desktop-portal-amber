Member Type Documentation

enum FileChooserPortal::PickerResponse

Possible return values of a dialog interaction. The values correspond to the response poperties of the calls in this class.

ConstantValueDescription
Amber::FileChooserPortal::Accepted0The dialog has been accepted
Amber::FileChooserPortal::Cancelled1The dialog has been cancelled
Amber::FileChooserPortal::Other2Something else has happened (e.g. a timeout)
